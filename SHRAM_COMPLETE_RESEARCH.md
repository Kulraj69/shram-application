## Executive Summary

**Shram AI** is an AI-powered desktop agent that **finds follow-ups and executes them automatically**. It monitors WhatsApp, Gmail, and Calendar to detect when responses are needed, drafts context-aware replies using on-device memory, and executes follow-ups with one-click through its desktop agent.

**Positioning**: "Never let a conversation go cold"

## Key Insights for Shram Application

### What Makes Shram Special

1. **Desktop agent execution** - Unlike most AI tools that stop at drafting
2. **WhatsApp integration** - Captures informal professional comms
3. **"No setup required"** - Works immediately out of the box
4. **Privacy-first** - On-device memory without cloud dependency

### Potential Concerns

1. **Limited track record** - Company founded 2024-2025
2. **Small team** - 2-10 employees
3. **Integrations incomplete** - User pain point
4. **Desktop-only** - Limits mobile-first users

### Moat Analysis

| Moat Component | Strength |
